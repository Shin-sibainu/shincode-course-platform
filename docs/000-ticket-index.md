## 開発の進め方

1. チケットは番号順に実施することを推奨（依存関係があるため）
2. 各チケット内の Todo リストを使用して進捗管理
3. 完了したタスクは `[ ]` を `[x]` に変更
4. ブランチ名は `feature/ticket-001-initial-setup` のような形式を推奨
